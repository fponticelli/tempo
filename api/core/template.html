module 'template'

A Template is a declarative representation of a View. The Template will be interpreted and realized into a View by invoking the render method.

types

RenderMethod

Function signature for the Template.render() method.

type RenderMethod<State, Query, Context> = (
  ctx: Context,
  state: State
) => View<State, Query>

Template

An object representing a Template that can be rendered into a View. 

interface Template<State, Query, Context> {
  render: RenderMethod<State, Query, Context>
}
