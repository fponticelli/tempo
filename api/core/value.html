module 'value'

All attributes and DOM values in a Tempo app are described by either literal values or values that are derived from the app current state.

types

DerivedOrLiteralValue

A value that can be either a literal or derived from a state.

type DerivedOrLiteralValue<State, Value> = Value | DerivedValue<State, Value>

DerivedValue

Describes a Value that can only be inferred from some other value State. 

type DerivedValue<State, Value> = (state: State) => Value | undefined
